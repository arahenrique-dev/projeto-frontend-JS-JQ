JQUERY

ESCREVA MENOS, FAÇA MAIS

É uma biblioteca Javascript. Criada for John Resig em 2006. Conhecida como “escreva menos, faça mais”, ela facilita desenvolver de forma rápida e otimiza a performance do produto.


A JQuery tem como objetivo simplificar o processo de desenvolvimento compactando diversas linhas de código em métodos que podem ser chamados em uma linha.

Sobre

Um dos recursos do JQuery é que ele permite que os erros sejam corrigidos de forma automática. Além disso, a sua execução é feita de forma cruzada entre as plataformas, de forma idêntica em diversos navegadores. Atualmente, ela é compatível com a maioria dos navegadores populares, funcionando com seletores CSS3


Recursos

Alguns recursos do JQuery:

-Manipulação CSS
-AJAX
-Manipulação do DOM/HTML
-Animações
